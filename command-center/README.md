# SG Media Command Center

The ultimate Godly-tier SaaS Command Center. Built to uncover business bottlenecks, run highly-focused sprints, and deploy AI-driven insights to scale.

## Architecture

This frontend is designed as a "Dark-First" premium UI, featuring:
- **Godly Theme:** Pure `#0A0A0C` midnight black, hyper-saturated mesh gradient orbs (`framer-motion`), and skeletal grids.
- **True Glassmorphism:** Cards use 40% black transparency with sharp `1px` inner white borders to simulate real glass edges.
- **Native Polish:** Custom pill scrollbars, spring physics (`framer-motion`) for hover states, and a Raycast-style `[CMD+K]` palette.
- **Agentic Core:** Designed to seamlessly integrate with an MCP (Model Context Protocol) backend.

## Tech Stack

- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Custom CSS (`index.css`)
- **Typography:** Plus Jakarta Sans & JetBrains Mono
- **Animations:** Framer Motion
- **Components:** Shadcn UI

## Getting Started

To run the application locally:

```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev
```

The application will run on `http://localhost:8080`.
